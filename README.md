# PROJETO PARA ESTUDOS DOS PIRATAS

Esse projeto é utilizado apenas para estudos e aplicação de conhecimentos em ReactJS.

## O que voce precisa ter instalado

para pdoer rodar esse projeto voce deve ter instalado em sua maquina o
[Node](https://nodejs.org/en/),[Npm](https://www.npmjs.com/) e o [Yarn](https://yarnpkg.com/)

## Baixar o projeto

Para baixar o projeto você precisa abrir o terminal na pasta que deseja baixar o projeto, e digitar o comando:

`git clone https://github.com/vitorhiro/piratas-project.git`

## Instalar o projeto

Assim que finalizar o download, abra o projeto no vscode, depois de abrir abra o terminal do vscode e digite
`yarn` para realizar a instação das dependências do pejeto.

## Rodar o projeto

Para rodar o projeto basta rodar o comando

`yarn start`

## O que foi utilizado no projeto

Esse projeto foi desenvolvido em [ReactJs](https://pt-br.reactjs.org/) utilizando o [Contex](https://pt-br.reactjs.org/docs/context.html) para armazenamentos de estados globais, o [react router dom](https://v5.reactrouter.com/web/guides/quick-start) para fazer as navegações de tela, e o [Style Components](https://styled-components.com/) para realizar a criação de components ja estilizados.
